## 자동차 경주 기능 요구사항 정리
### 요구사항 분석
- 경주에 참여할 자동차의 이름을 입력받는다.
- 각 자동차의 정보는 쉼표로 구분한다.
- 자동차의 이름은 5자 이하만 가능하다.
- 게임 시도 횟수를 입력받는다.
- 각 횟수, 자동차마다 랜덤으로 값을 생성해야한다.
- 자동차가 전진하는 조건은 랜덤으로 생성한 값이 4 이상일 경우이다.
- 횟수가 모두 끝나면, 우승자를 표시한다.
- 우승자는 가장 멀리 간 사람이다.
- 우승자는 한 명 이상일수 있다.
- 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시킨다.
